About Us

We are Shaked Hazon and Ron Michaeli, students at Ben-Gurion University.

Hoping you are enjoying our coolest PacMan game (tough ghosts, ah?)

In this nice little game we used several libraries, such as jQuery.
We used jQuery mainly to validate user input on the SignUp and Login forms.

We also used a template for our awesome top navigation bar. Thanks Internet!

The major difficulty we encountered while doing this task was understanding how to create simple layouts using HTML and design them with CSS3.
For example - how to center a div inside another div. Turns out it's one of the hardest things to do in this world.

Well, that's it from us...

Have fun!
